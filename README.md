# Hoi, ik ben Jeroen 👋

Founder van **[SparkOne](https://sparkone.nl)**. Ik help mensen en organisaties
slimmer werken met AI, en bouw zelf de tools die dat laten zien. Veel van mijn
werk zit op het snijvlak van **AI, innovatie en de digitale overheid**.

## Waar ik mee bezig ben

- **SparkOne** — praktische AI-innovatie, plus de serie *"Zo gebruik ik AI"*
  waarin ik laat zien hoe ik zelf met AI werk.
- **GovTech & trendverkenning** — verkennen welke nieuwe technologieën ertoe
  doen voor de Nederlandse digitale overheid, en dat toegankelijk maken.
- **Bouwen met AI-agents** — ik zet Claude Code in om echte, werkende demo's te
  bouwen zonder zware buildstap.

## Uitgelicht

[**exploratie-hub**](https://github.com/jeroen-sparkone/exploratie-hub) —
werkende demo van een Exploratie Hub: één plek om te zien welke technologieën
binnen de NL digitale overheid onderzocht worden, wie dat doet en hoe volwassen
ze zijn. Trendroos-taxonomie + Novum-funnel, volledig client-side.
👉 [Live demo](https://jeroen-sparkone.github.io/exploratie-hub/)
